HR Analytics Dashboard | Power BI
Project Overview
This project focuses on analyzing employee workforce data to provide meaningful insights into headcount, attrition, salary distribution, and workforce composition.
The dashboard is built using Power BI and is designed to support HR and management decision-making through interactive and visual reporting.
